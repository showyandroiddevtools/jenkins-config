This repo contains configuration files for Jenkins.

To get started with a new Jenkins setup...

1) Clone this repo (into $JENKINS_ROOT)
2) Download jenkins.war into $JENKINS_ROOT/bin/
   [optional] Also download jenkins-cli.war 
3) mkdir $JENKINS_ROOT/data, then install...
   android-ndk/
   android-sdk/
   gradle
   jdk1.6
   jdk1.7
4) mkdir $JENKINS_ROOT/log
